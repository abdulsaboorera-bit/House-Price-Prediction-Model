## 📌 Project Overview

This project builds a **Machine Learning model** that predicts the **price of a house** based on three important features:

* Age of the house
* Area of the house (square feet)
* Number of bedrooms

The model learns patterns from housing data and estimates the expected price of a property.

---

## 🎯 Input Features

The model uses the following inputs:

| Feature  | Description                           |
| -------- | ------------------------------------- |
| Age      | How old the house is (in years)       |
| Area     | Total area of the house (square feet) |
| Bedrooms | Number of bedrooms in the house       |

Example input:

Age = 10 years
Area = 2000 sq ft
Bedrooms = 3

The model will predict the **estimated house price**.

---

## 🧠 Machine Learning Model

This project uses **Linear Regression** to predict house prices based on the relationship between the input features and the price.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Google Colab / Jupyter Notebook

---

## 📊 Workflow

1. Load the dataset
2. Prepare input features (Age, Area, Bedrooms)
3. Train the Linear Regression model
4. Evaluate model accuracy
5. Predict house prices
